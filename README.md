# HTML to WordPress Page

A WordPress plugin that allows you to create standalone HTML pages without the WordPress theme header/footer. Perfect for uploading AI-generated HTML code.

## Features

- Simple admin interface to manage HTML pages
- Upload HTML files or paste HTML directly
- Drag and drop HTML files into the editor
- Auto-generate slugs from titles
- Pages render pure HTML without any WordPress theme elements
- Public URLs at `yoursite.com/html/page-slug`

## Installation

1. Upload the `html-to-wordpress-page` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **HTML Pages** in the admin menu

## Usage

1. Navigate to **WordPress Admin > HTML Pages > Add New**
2. Enter a title and slug for your page
3. Either:
   - Upload an HTML file using the file input
   - Drag and drop an HTML file into the textarea
   - Paste HTML code directly
4. Click **Create Page**
5. Your page will be available at `yoursite.com/html/your-slug`

## Requirements

- WordPress 5.0 or higher
- PHP 7.2 or higher
